# API REST Node.js

API REST desenvolvida com Node.js e TypeScript, focada no aprendizado de conceitos fundamentais de backend e boas práticas.

Projeto criado durante estudos práticos de desenvolvimento de APIs.

---

## Tecnologias
- Node.js
- TypeScript
- Fastify
- Knex.js
- SQLite
- Zod

---

## Funcionalidades
- Criação de rotas REST
- Cadastro e listagem de dados
- Validação de requisições
- Integração com banco de dados

---

## Como rodar o projeto

```bash
# instalar dependências
npm install

# rodar migrations (se houver)
npm run knex:migrate

# iniciar o servidor
npm run dev
